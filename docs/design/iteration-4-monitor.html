5. 边缘情况 & 空状态

📭 无供应商配置

展示空状态引导：「还没有配置供应商呢～ 点击上方"供应商管理"添加第一个供应商吧 ✨」
配合一个简洁的插图 + CTA 按钮跳转到供应商 CRUD 页面。

🔑 所有供应商无 Key

所有供应商卡片显示 Unknown 状态，顶部展示全局提示条：「所有供应商暂无可用 Key，请在供应商管理中配置 API Key 后巡检将自动启动。」

📈 用量报告无数据

折线图区域展示空状态：「所选时间范围内暂无用量数据」
日期选择器仍然可用，方便用户切换到有数据的时间范围。

⏱ 手动巡检进行中

「手动巡检」按钮变为 loading 状态（旋转图标 + 文字变为"巡检中…"），禁用重复点击。
完成后按钮恢复 + toast 通知「巡检完成，X 个供应商已更新」。

📱 移动端适配

健康卡片：3 列 → 1 列堆叠
用量图表：全宽展示，控件行换行
时间线：左侧时间轴保留，详情自适应折行
